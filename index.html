My Projects

Real time COVID-19 Tracking and Vaccination slots notifier Android Application

June 2021 - June 2021

This application has two features.

  1. Shows real-time COVID-19 stats nationwise as well as statewise in terms total cases, total recovered, total deceased, last 24 hours data.
    • In the backend we are fetching data using REST APIs provided by our Indian Govt. for developers. We are fetching data in Background thread so that our main UI doesn't get blocked.
  2. Provides an interface where our user can enter details (Email, State, District). By using this feature our users can check the availabilty of vaccine slots in their area. This app will ensure that once a slot is available in our user's location it will notify them by sending a mail. Furthermore, this app ensures that it will keep notifying our users after a fixed interval of time unless and until our users press the cancel button.
    • In the backend we are fetching data using REST APIs provided by our Indian Govt. and then we are parsing data which is available in JSON as per our need. In order to send mail we are using SMTP mail server. In order to keep sending mail after a fixed interval of time, we are using Work Manager. Work Manager is an Android Jetpack library for deferrable, reliable background work.

Techstack Used: Android Studio, XML, Java, Work Manager, JSON, REST APIs, List View, Array Adapters, Threading.

Report Card Python Automation Project

July 2021 - August 2021

All about this project:

  • This is Automation Project whose source code is written by me in Python. I made this Project during Intern hiring Challenge organized by Wisdom Tests (Autrailian Edu - Tech Company and I made it to top 20(till final round) among many participants accross the world) .
  • Requirement - I was required to develop an executable file, which once clicked generates required report cards of corresponding students in PDF format and the design of PDFs must match IMO till section 2.
  • Input - As an input I was provided with an excel file where the data of all the students were present (Headers - Student Name , Student Roll No. , Total Questions , Reponses Of All Students etc), Pictures of all students, logo of Internation Mathematics Olympiad (IMO), background image of pdf.
  • Working - I used Python 2.7 to achieve the target. I used Openpyxl library to read data from Excel file, Reportlab library to generate PDF, Canvas for designing, Metaplotlib for Bar Graphs and Pyinstaller to bundle Python source code file to executable file.

Techstack Used: Python 2.7, Reportlab library, Metaplotlib library, Openpyxl library, Canvas, Pyinstaller.
